# 🪑 NordForm — Furniture AR Visualizer

An interactive 3D and Augmented Reality (AR) experience for exploring and customizing a furniture product directly in the browser.

---

## 🔗 Live Demo

👉 https://furniture-ar-visualizer.onrender.com/

---

## ✨ Features

* 🎯 Interactive 3D chair (rotate, zoom)
* 🎨 Fabric color customization (Grey / Beige / Olive)
* 🌗 Ambiance modes (Neutral / Warm / Dark)
* 📱 AR preview on mobile (WebXR / Scene Viewer / Quick Look)

---

## 📌 Notes

* AR works only on mobile devices and deployed version (HTTPS required)
* Current model uses a single mesh → color applies to entire chair
